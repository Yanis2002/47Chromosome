# Автоматическая генерация контента

Скрипт `generate-content.js` автоматически создает JSON файлы и обновляет код из файлов в папках.

## Использование

Просто запустите:

```bash
node generate-content.js
```

## Что делает скрипт

1. **Музыка** (`docs/music/`)
   - Сканирует все аудио файлы (.mp3, .wav, .ogg, .flac, .m4a)
   - Автоматически обновляет массив `localMusic` в `docs/script.js`
   - Создает названия треков из имен файлов

2. **Фото** (`docs/photo/`)
   - Сканирует все изображения (.jpg, .jpeg, .png, .webp, .gif, .svg)
   - Автоматически создает/обновляет `docs/photo/list.json`
   - Создает описания из имен файлов

3. **Видео** (`docs/video/`)
   - Сканирует все видео файлы (.mp4, .webm, .ogg, .mov, .avi)
   - Автоматически обновляет массив `localVideos` в `docs/script.js`
   - Создает названия из имен файлов

## Пример работы

1. Добавьте новые файлы в соответствующие папки:
   ```bash
   # Добавьте музыку
   cp ~/Downloads/new-track.mp3 docs/music/
   
   # Добавьте фото
   cp ~/Pictures/my-photo.webp docs/photo/
   ```

2. Запустите скрипт:
   ```bash
   node generate-content.js
   ```

3. Скрипт автоматически:
   - Найдет все новые файлы
   - Обновит JSON файлы
   - Обновит код в script.js

4. Закоммитьте изменения:
   ```bash
   git add .
   git commit -m "Add new music and photos"
   git push
   ```

## Формат имен файлов

Скрипт автоматически создает названия из имен файлов:
- `Artist - Song Name.mp3` → `Artist - Song Name`
- `my_photo_2024.webp` → `my photo 2024`
- `[MP3DOWNLOAD.TO] Track.mp3` → `Track` (убирает префиксы в скобках)

## Требования

- Node.js (любая версия)
- Файлы должны быть в правильных папках:
  - `docs/music/` - для музыки
  - `docs/photo/` - для фото
  - `docs/video/` - для локальных видео

## Примечания

- Скрипт не удаляет старые записи, только добавляет новые
- Если файл уже есть в списке, он не будет дублирован
- Для YouTube видео используйте `docs/video/youtube.json` или `docs/video/links.txt` вручную
