# 47Chromosome - Maidcore Aesthetic Site

Личный сайт в стиле Maidcore с разделами для эстетики, аудио, видео, фото, библиотеки и магазина.

**Все файлы сайта находятся в папке `docs/`**

## Настройка GitHub Pages

1. Перейдите в Settings → Pages
2. Выберите ветку `main` и папку `/docs`
3. Сайт будет доступен по адресу: `https://yanis2002.github.io/47Chromosome/`

## Структура

- `docs/` - все файлы сайта для GitHub Pages
- `examples.js` - примеры использования функций
- `README.md` - документация

Подробная документация в файлах внутри `docs/`.
