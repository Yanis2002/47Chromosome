# 47Chromosome - Личный сайт

Личный сайт с разделами для эстетики, аудио, видео, фото, библиотеки и магазина.

## Запуск

### Локально (без портов)

Просто откройте `index.html` в браузере - двойной клик или перетащите файл в окно браузера.

### С локальным сервером (опционально)

Если нужен локальный сервер:

```bash
# Python 3
python3 -m http.server 8000

# Node.js
npx http-server
```

Затем откройте `http://localhost:8000`

## GitHub Pages

1. Перейдите в Settings → Pages
2. Выберите ветку `main` и папку `/ (root)`
3. Сайт будет доступен по адресу: `https://yanis2002.github.io/47Chromosome/`

## Структура

- `index.html` - главная страница
- `styles.css` - стили
- `script.js` - JavaScript
- `music/` - папка с музыкой
- `photo/` - папка с фотографиями
- `video/` - папка с видео и YouTube ссылками
- `generate-content.js` - скрипт для автоматической генерации списков

## Автоматическая генерация контента

Запустите `node generate-content.js` для автоматического обновления списков музыки, фото и видео из папок.
