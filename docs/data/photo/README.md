# Папка для фотографий

## Поддерживаемые форматы

- **WebP** (.webp) - рекомендуется для веба
- **JPEG** (.jpg, .jpeg)
- **PNG** (.png)
- **GIF** (.gif)
- **SVG** (.svg)

## Как добавить фотографии

### Способ 1: JSON файл (рекомендуется)

1. Поместите ваши фотографии в эту папку (любой из поддерживаемых форматов)
2. Отредактируйте файл `list.json` и добавьте ваши фото:

```json
[
  {
    "src": "photo/my-photo1.webp",
    "alt": "Описание фото 1"
  },
  {
    "src": "photo/my-photo2.jpg",
    "alt": "Описание фото 2"
  },
  {
    "src": "photo/my-photo3.png",
    "alt": "Описание фото 3"
  }
]
```

**Важно:** Указывайте путь относительно папки `docs/`, например `photo/файл.webp`

Фотографии будут автоматически загружены на сайт при открытии страницы.

### Способ 2: Через код

Добавьте фото в функцию `loadLocalPhotos()` в `script.js`:

```javascript
const localPhotos = [
    { src: 'photo/my-photo.webp', alt: 'Описание' },
    { src: 'photo/my-photo.jpg', alt: 'Описание' }
];
```
