# Папка для музыки

Поместите ваши аудио файлы (MP3, OGG, WAV и т.д.) в эту папку.

Затем добавьте их в `script.js` в функцию `loadLocalMusic()`:

```javascript
const localMusic = [
    { 
        src: 'music/ваш-файл.mp3', 
        title: 'Название трека', 
        duration: '0:00' 
    }
];
```

Файлы будут автоматически загружены при открытии сайта.
